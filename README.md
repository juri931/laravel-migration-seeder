Creiamo una tabella trains e relativa Migration

Ogni treno dovrà avere:

-   Azienda
-   Stazione di partenza
-   Stazione di arrivo
-   Orario di partenza
-   Orario di arrivo
-   Codice Treno
-   Numero Carrozze
-   In orario
-   Cancellato

Oltre alla migration di creazione della tabella aggiungente una migration di update con rollback funzionante
